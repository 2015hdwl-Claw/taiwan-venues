## 💡 給 Grok 的額外建議

如果 Grok 需要更多指導，可以補充：

```
額外要求：
1. 照片搜集：
   - 優先尋找有官方照片的場地
   - 照片 URL 必須是直接連結（不是網頁）
   - 如果找不到照片，該欄位留空

2. 布置圖搜集：
   - 尋找場地的平面圖、座位配置圖
   - 官方網站通常會提供下載連結
   - 如果找不到，該欄位留空

3. 場地別判斷：
   - 根據場地的主要用途分類
   - 如果場地有多種用途，選擇最主要的
   - 例如：飯店裡的會議室歸類為「飯店場地」

4. 可租用時間說明：
   - 收集預約規則
   - 注意假日是否開放
   - 是否需要提前多久預約

5. 資料驗證：
   - 確保電話格式正確
   - 地址可對應到實際地點
   - 價格資訊合理
   - 照片連結有效
```
