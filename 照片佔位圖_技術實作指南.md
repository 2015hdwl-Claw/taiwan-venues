### 步驟 2：修改渲染邏輯

**原本的程式碼**：
```javascript
<div class="venue-image">
    ${venue.images?.main 
        ? `<img src="${venue.images.main}" alt="${venue.name}" loading="lazy">`
        : `<div class="venue-image-placeholder">🏢</div>`
    }
</div>
```

**修改為**：
```javascript
<div class="venue-image">
    ${venue.images?.main 
        ? `<img src="${venue.images.main}" alt="${venue.name}" loading="lazy">`
        : `<div class="venue-image-placeholder" 
             style="background: ${getVenueGradient(venue.venueType)}"
             data-type="${venue.venueType || '會議場地'}">
             <div class="placeholder-icon">${getVenueIcon(venue.venueType)}</div>
             <div class="placeholder-type">${venue.venueType || '會議場地'}</div>
             <div class="placeholder-text">照片籌備中</div>
           </div>`
    }
</div>
```
