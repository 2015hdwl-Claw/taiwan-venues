## 📋 每個會議室需要收集的資訊

### ✅ 必要資訊

1. **會議室名稱**
   - 完整名稱（例如：101會議室、大會堂）

2. **真實照片**
   - 會議室內部照片 URL
   - 必須是真實照片，不能用 Unsplash 或示意圖

3. **場地大小**
   - 坪數或平方公尺
   - 必須是準確數據

4. **容納人數**
   - 劇院型：多少人
   - 課桌式：多少人
   - 宴會型：多少人
   - 空場：多少人

5. **價格**
   - 半日價（上午或下午）
   - 全日價
   - 時段說明（例如：平日/假日）

6. **設備清單**
   - 投影機、音響、麥克風、白板、WiFi 等

7. **聯絡方式**
   - 電話
   - Email
   - 官網 URL

---

### ✅ 資料來源標記

每個會議室必須包含：

```json
{
  "dataSource": "official_website",
  "dataSourceUrl": "https://...",
  "dataCollectedAt": "2026-02-28T12:00:00Z",
  "verified": true,
  "collectionMethod": "deep_crawl"
}
```

## 🔍 爬蟲方法

### 方法 1：官方網站爬蟲
- 訪問官網的會議室介紹頁面
- 提取每個會議室的詳細資訊
- 下載真實照片

### 方法 2：搜尋引擎輔助
- 如果官網結構複雜，使用搜尋找到會議室頁面
- 確保資料來自官方來源

### 方法 3：電話查證
- 如果網站資訊不完整，需要電話查證
- 記錄查證時間和對象
