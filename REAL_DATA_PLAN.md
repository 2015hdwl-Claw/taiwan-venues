## 📝 收集流程

### Step 1: 驗證現有資料
```
1. 檢查場地是否真實存在
2. 驗證聯絡電話/Email
3. 測試照片 URL 是否有效
4. 標記資料狀態
```

### Step 2: 收集新資料
```
1. 訪問官網
2. 提取場地資訊
3. 下載真實照片
4. 記錄來源 URL
```

### Step 3: 整理資料
```
1. 統一格式
2. 驗證完整性
3. 匯入資料庫
```

| 欄位 | 驗證方式 |

| Email | 格式正確 + 官網確認 |
| 照片 | URL 可訪問 (200 OK) |
