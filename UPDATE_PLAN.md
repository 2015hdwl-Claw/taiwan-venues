# 更新版台灣活動場地資料庫 - 新功能規劃

## 🆕 新增欄位

### 1. 場地別（Venue Category）
```
- 飯店場地
- 婚宴場地
- 會議場地
- 機關場地
- 展演場地
- 運動場地
- 戶外場地
- 共享空間
- 教育場地
- 其他
```

### 2. 廳別（Room Name）
- 更清楚的命名：會議室名稱/廳別
- 例如：101會議室、宴會廳A、展覽館B

### 3. 可租用時間（Available Dates）
- 使用日曆系統
- 標記可用/不可用日期
- 顯示已預約時段

### 4. 場地照片（Photos）
- 支援多張照片
- URL 或本地上傳
- 照片輪播展示

### 5. 場地布置圖（Layout Images）
- 平面圖
- 座位配置圖
- 設備配置圖

### 6. 日曆視圖（Calendar View）
- 月曆顯示
- 選擇日期查看可用場地
- 場地預約管理

---

## 📊 更新後的 CSV 格式（24個欄位）

```
場地名稱,場地別,廳別,類型,縣市,地址,聯絡人,電話,電子郵件,每小時費用,半天費用,全天費用,最大容納(空場),最大容納(劇院型),最大容納(教室型),平日時間,假日時間,設備費用,場地照片,場地布置圖,可租用時間,備註,活動規模,建立日期
```

### 新增欄位說明：

1. **場地別**（必填）
   - 飯店場地/婚宴場地/會議場地/機關場地/展演場地/運動場地/戶外場地/共享空間/教育場地/其他

2. **場地照片**（選填）
   - 照片 URL，多張用分號分隔
   - 例如：http://example.com/photo1.jpg;http://example.com/photo2.jpg

3. **場地布置圖**（選填）
   - 布置圖 URL，多張用分號分隔

4. **可租用時間**（選填）
   - JSON 格式或簡單描述
   - 例如：{"available":["2026-03-01","2026-03-02"],"unavailable":["2026-03-15"]}
