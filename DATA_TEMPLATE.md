# 場地資料收集範本

## 📋 必填欄位

### 基本資訊
- **場地名稱**: 完整的場地名稱
- **廳別**: 具體的廳別/房間名稱
- **場地類型**: 飯店場地/會議場地/展演場地/機關場地/戶外場地
- **縣市**: 台灣22縣市
- **地址**: 完整地址

### 聯絡資訊
- **聯絡人**: 場地租借窗口
- **電話**: 格式 02-1234-5678
- **Email**: 正確的 Email 格式

### 費用資訊
- **每小時費用**: 數字（可留空）
- **半天費用**: 數字（可留空）
- **全天費用**: 數字（可留空）

### 容納資訊
- **最大容納(空場)**: 數字
- **最大容納(劇院型)**: 數字（可留空）
- **最大容納(教室型)**: 數字（可留空）

### 其他資訊
- **使用時間**: 平日/假日可使用時間
- **設備**: 提供的設備
- **備註**: 其他重要資訊

---

## 📸 照片規格

### 主照片
- **來源**: 必須從官網實際取得
- **格式**: JPG/PNG
- **URL**: 必須可正常訪問（200 OK）
- **尺寸**: 建議 1200x800 以上

### 畫廊照片
- 可選，建議 2-5 張
- 展示場地不同角度

### 平面圖
- 可選，座位配置圖

---

## ✅ 驗證標準

每筆資料必須通過以下驗證：

| 項目 | 驗證方式 | 標準 |
|------|----------|------|
| 場地存在 | 官網確認 | ✅ |
| 地址正確 | Google Maps | ✅ |
| 電話有效 | 實際撥打/官網 | ✅ |
| Email 格式 | 格式檢查 | 無中文 |
| 照片 URL | HTTP 請求 | 200 OK |
| 資料來源 | 記錄來源 | 官網 URL |
