# 台灣活動規劃大師 - 產品規劃書

> 從「場地媒合平台」進化為「活動規劃入口網站」

---

## 🎯 產品定位

**台灣活動規劃大師** 是一個一站式活動規劃入口網站，協助活動主辦人從需求發想到供應商媒合，完成整個活動的規劃與執行。

### 核心價值
- **一站式服務**：場地、餐飲、製作物、執行人員全部搞定
- **智能推薦**：學習 TikTok 演算法，根據用戶行為精準推薦
- **AI 視覺化**：用 AI 繪圖讓用戶預見活動畫面，提高轉換率
- **供應商媒合**：自動將需求發送給合適的供應商

## 🏗️ 功能模組

### Phase 1：場地與需求收集（現有基礎擴展）

#### 1.1 智能需求嚮導
```
用戶輸入 → 系統引導 → 偏好學習 → 智能推薦
```

**需求收集流程**：
1. 活動類型（會議/尾牙/婚禮/展覽/派對...）
2. 預估人數（自動帶入場地容納篩選）
3. 日期時間（自動排除已預約場地）
4. 地區偏好（縣市 + 交通需求）
5. 預算範圍（總預算或各項預算）
6. 風格偏好（正式/活潑/溫馨/科技感...）
7. 特殊需求（餐飲、佈置、錄影...）

#### 1.2 TikTok 式推薦引擎
**行為追蹤**：
- 點擊的場地類型
- 瀏覽時間長度
- 比較的場地組合
- 篩選條件調整
- 放棄的選項

**推薦邏輯**：
```python
用戶畫像 = {
    "活動類型偏好": 權重計算,
    "價格敏感度": 高/中/低,
    "地區偏好": 排序,
    "風格偏好": 標籤,
    "轉換機率": 預測分數
}

→ 動態調整推薦順序
→ 個人化內容呈現
```

### Phase 3：AI 視覺化

#### 3.1 AI 活動場景生成

**流程**：
```
用戶需求 + 選擇項目 → AI Prompt 生成 → 圖片生成 → 展示給用戶
```

**Prompt 範例**：
```
A corporate year-end party at Grand Hyatt Taipei ballroom,
300 guests, round tables with elegant floral centerpieces,
stage with LED backdrop showing company logo,
warm lighting, professional event photography style,
4K, photorealistic
```

**應用場景**：
- 場地選擇時：生成不同場地的活動效果圖
- 佈置規劃時：生成佈置示意圖
- 提案展示時：生成完整的活動視覺

#### 3.2 AI 輔助規劃

- **預算分配建議**：根據活動類型建議各項目預算比例
- **時間軸規劃**：自動生成活動流程時間表
- **供應商媒合**：根據需求自動推薦最佳組合

### Phase 4：訂單與媒合系統

#### 4.1 訂單管理

```
需求收集 → 方案生成 → 用戶確認 → 發送供應商 → 報價比較 → 確認訂單
```

**訂單狀態**：
- 草稿（規劃中）
- 詢價中（已發送供應商）
- 報價完成（收到報價）
- 已確認（用戶選擇）
- 執行中（活動進行）
- 已完成（活動結束）
- 評價（用戶評分）

#### 4.2 供應商後台

**功能**：
- 接收媒合需求
- 提供報價
- 確認接單
- 行程管理
- 收款管理

## 🔧 技術架構

### 前端
- **框架**：React / Vue.js / Next.js
- **UI**：Tailwind CSS / shadcn/ui
- **狀態管理**：Zustand / Redux
- **AI 圖片**：整合 Stable Diffusion / DALL-E API

### 後端
- **API**：Node.js / Python FastAPI
- **資料庫**：PostgreSQL（關聯資料） + MongoDB（彈性資料）
- **搜尋**：Elasticsearch / Meilisearch
- **推薦引擎**：Python + scikit-learn / TensorFlow

### 基礎設施
- **部署**：Vercel / AWS / GCP
- **CDN**：CloudFlare
- **圖片儲存**：Cloudinary / S3
- **即時通訊**：WebSocket（供應商即時通知）
