# 台灣活動規劃大師

> 🎉 辦活動，找大師就對了！

## 📋 簡介

**台灣活動規劃大師** 是一個一站式活動規劃入口網站，協助活動主辦人從需求發想到供應商媒合，完成整個活動的規劃與執行。

## 🎯 四大服務模組

| 服務 | 圖標 | 狀態 | 說明 |
|------|------|------|------|
| **活動場地** | 🏢 | ✅ 已上線 | 會議室、宴會廳、展演空間（281 個場地） |
| **活動餐飲** | 🍽️ | 🚧 即將推出 | 茶點、外燴、自助餐、桌餐 |
| **活動製作物** | 🎨 | 🚧 即將推出 | 背板、佈置、禮品、印刷 |
| **活動執行** | 🎬 | 🚧 即將推出 | 主持人、樂隊、攝影、服務人員 |

## 🌐 網站

https://2015hdwl-claw.github.io/taiwan-venues/

## 🎯 活動規模分類

- **微型活動**：50人以下
- **小型活動**：50-100人
- **中型活動**：100-200人
- **中大型活動**：200-500人
- **大型活動**：500-1000人
- **超大型活動**：1000-5000人
- **巨型活動**：5000人以上

## 📊 目前資料狀態

- **總場地數**：281 筆
- **整體完整度**：89.3%
- **資料充足縣市**：13 個（≥10 筆）
- **待收集縣市**：5 個（基隆、嘉義市/縣、雲林、台東）

## 🚀 功能特色

- ✅ 響應式設計，支援手機、平板、電腦
- ✅ 活動類型選擇器（公司會議、尾牙春酒、婚禮、展覽等）
- ✅ 場地搜尋與篩選（縣市、規模）
- ✅ 照片畫廊（主照片 + 畫廊 + 平面圖）
- ✅ 場地比較功能
- ✅ 服務類型入口框架
