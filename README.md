# 台灣活動場地資料庫

## 📋 簡介

這是一個收集全台灣活動場地的網頁系統，可以根據活動規模快速搜尋和篩選適合的場地。

## 🎯 活動規模分類

- **微型活動**：50人以下
- **小型活動**：50-100人
- **中型活動**：100-200人
- **中大型活動**：200-500人
- **大型活動**：500-1000人
- **超大型活動**：1000-5000人
- **巨型活動**：5000人以上

## 📊 場地資訊

每個場地包含以下資訊：

### 基本資訊
- 場地名稱
- 廳別/房間名稱
- 場地類型
- 縣市
- 詳細地址

### 聯絡資訊
- 聯絡人
- 聯絡電話
- 電子郵件

### 費用資訊
- 每小時費用
- 半天費用
- 全天費用

### 容納資訊
- 最大容納人數（空場/不放椅子）
- 最大容納人數（劇院型）
- 最大容納人數（教室型）

### 使用資訊
- 平日可使用時間
- 假日可使用時間
- 設備與費用
- 備註

## 🚀 使用方式

### 1. 開啟網頁

直接用瀏覽器開啟 `index.html` 檔案即可使用。

### 2. 新增場地

1. 點擊「新增場地」按鈕
2. 填寫場地資訊
3. 點擊「新增場地」儲存

### 3. 搜尋場地

- 在搜尋框輸入關鍵字（場地名稱、地點等）
- 使用篩選器選擇活動規模、縣市、場地類型

### 4. 查看詳情

點擊場地卡片可查看完整資訊。

### 5. 匯出/匯入資料

- **匯出 CSV**：將所有場地資料匯出為 CSV 檔案
- **匯出 JSON**：將所有場地資料匯出為 JSON 檔案
- **匯入資料**：可匯入 CSV 或 JSON 格式的場地資料

## 💾 資料儲存

所有資料儲存在瀏覽器的 LocalStorage 中，關閉瀏覽器後資料仍會保留。

**重要**：清除瀏覽器資料會導致場地資料消失，建議定期匯出備份。

## 🔧 功能特色

- ✅ 響應式設計，支援手機、平板、電腦
- ✅ 即時搜尋與篩選
- ✅ 按活動規模自動分類
- ✅ 匯出 CSV 和 JSON 格式
- ✅ 匯入現有資料
- ✅ 詳細的場地資訊展示
- ✅ 美觀的使用者介面

## 📊 統計功能

首頁會顯示各種活動規模的場地數量統計，方便快速了解資料庫狀況。

## 🌐 部署方式

### 本地使用
直接開啟 `index.html` 即可使用。

### 網站部署
將所有檔案上傳到任何網頁伺服器即可。

### GitHub Pages
1. 將檔案上傳到 GitHub 儲存庫
2. 在儲存庫設定中啟用 GitHub Pages
3. 選擇分支和資料夾
4. 即可透過 GitHub Pages 網址存取

## 🐛 已知問題

- 匯入 CSV 時，欄位中的逗號可能會造成解析錯誤（建議使用 JSON 格式）
- LocalStorage 有容量限制（約 5-10MB）
