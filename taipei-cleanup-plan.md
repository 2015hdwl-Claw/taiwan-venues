### 任務 1: 標記已歇業場地（週六 3/8）
**時間**: 30 分鐘  
**目標**: 將 21 個已歇業場地標記為「下架」  
**方法**:
```bash
cd /root/.openclaw/workspace/taiwan-venues
python3 << 'EOF'
import json

with open('venues-all-cities.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

# 已歇業場地 ID 列表
closed_ids = [1048, 1055, 1057, 1059, 1065, 1066, 1069, 1073, 1074, 1080, 
              1084, 1092, 1104, 1396, 1397, 1398, 1399, 1400, 1401, 1402, 1403]

updated = 0
for v in data:
    if v.get('id') in closed_ids and v.get('status') != '下架':
        v['status'] = '下架'
        v['statusChangedAt'] = '2026-03-08T00:00:00.000Z'
        v['statusChangeReason'] = '已歇業/結束營業'
        updated += 1

with open('venues-all-cities.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

print(f'已標記 {updated} 個已歇業場地為「下架」')
EOF
```
