## ⚠️ 學到的教訓

### 1. status 必須是「上架」
```javascript
// ❌ 錯誤
v.status = 'active';

// ✅ 正確
v.status = '上架';
```

### 2. 照片不用維基百科
```javascript
// ❌ 錯誤
main: 'https://upload.wikimedia.org/wikipedia/...'

// ✅ 正確
main: 'https://官網/照片.jpg'
```

### 3. gallery 必須是陣列
```javascript
// ❌ 錯誤
gallery: "url1\nurl2"

// ✅ 正確
gallery: ["url1", "url2"]
```
