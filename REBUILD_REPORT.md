## 🚀 使用方式

### 給終端用戶
1. 訪問 https://taiwan-venues-new.vercel.app
2. 使用搜尋框輸入關鍵字
3. 使用篩選器縮小範圍
4. 點擊場地卡片查看詳情
5. 聯繫場地或訪問官網

### 給開發者
```bash
# 本地開發
cd taiwan-venues-new
python3 -m http.server 8080
# 訪問 http://localhost:8080

# 部署到 Vercel
vercel --prod
```
