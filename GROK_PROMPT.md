# Grok 提示詞：收集台灣活動場地資料庫

## 🎯 任務目標

請幫我收集全台灣可以用於舉辦活動的場地資訊，目標是建立一個完整的場地資料庫。這些資料將用於活動規劃和場地媒合。

## 📊 需要收集的場地類型

### 按活動規模分類：
1. **微型活動**（50人以下）：會議室、共享空間、咖啡廳包場
2. **小型活動**（50-100人）：小型會議廳、教室、社區活動中心
3. **中型活動**（100-200人）：中型會議廳、展演空間
4. **中大型活動**（200-500人）：大型會議廳、體育館、展演中心
5. **大型活動**（500-1000人）：體育場館、大型展演廳
6. **超大型活動**（1000-5000人）：體育場、展覽館、大型會議中心
7. **巨型活動**（5000人以上）：體育場、戶外場地

### 場地類型包括：
- 會議室/會議中心
- 活動中心/社區中心
- 展演空間/展覽館
- 體育館/體育場
- 教室/培訓中心
- 戶外場地
- 飯店會議室
- 咖啡廳/餐廳包場
- 共享工作空間
- 藝文中心

- "會議中心 [縣市名]"

```
台北市、新北市、桃園市、台中市、台南市、高雄市
基隆市、新竹市、嘉義市
新竹縣、苗栗縣、彰化縣、南投縣、雲林縣、嘉義縣、屏東縣

```

### 搜尋來源：
1. **官方網站**：
   - 各縣市政府官網（場地租借頁面）
   - 各級學校場地租借
   - 國父紀念館、中正紀念堂等
   - 各文化中心、社教館

2. **民營場地**：
   - 商業會議中心
   - 飯店會議室
   - 共享工作空間
   - 展覽公司
   - 活動場地平台（如 Accupass、KKTix 等）

3. **特殊場地**：
   - 大學院校場地
   - 社區活動中心
   - 宗教團體場地
   - 非營利組織場地

## 📋 必填欄位（CSV 格式）

### CSV 欄位順序：
```
場地名稱,廳別,類型,縣市,地址,聯絡人,電話,電子郵件,每小時費用,半天費用,全天費用,最大容納(空場),最大容納(劇院型),最大容納(教室型),平日時間,假日時間,設備費用,備註,活動規模
```

### 欄位說明：

1. **場地名稱**（必填）
   - 場地的主要名稱
   - 例如：「台北國際會議中心」

2. **廳別**（選填）
   - 具體的廳別或房間名稱
   - 例如：「101會議室」、「A廳」

3. **類型**（必填）
   - 選項：會議室、活動中心、展演空間、體育館、教室、戶外場地、飯店會議室、咖啡廳、共享空間

4. **縣市**（必填）
   - 台灣22縣市之一

5. **地址**（必填）
   - 完整地址，包含行政區
   - 例如：「台北市信義區信義路五段1號」

6. **聯絡人**（選填）
   - 場地租借的聯絡人姓名或單位

7. **電話**（選填）
   - 聯絡電話，格式：02-1234-5678

8. **電子郵件**（選填）
   - 聯絡 Email

9. **每小時費用**（選填）
   - 數字，不含貨幣符號
   - 例如：1500

10. **半天費用**（選填）
    - 數字，通常是 3-4 小時
    - 例如：5000

11. **全天費用**（選填）
    - 數字，通常是 8 小時
    - 例如：8000

12. **最大容納(空場)**（必填）
    - 不放椅子的最大容納人數
    - 數字

13. **最大容納(劇院型)**（選填）
    - 劇院式座位（只有椅子）的最大容納人數
    - 數字

14. **最大容納(教室型)**（選填）
    - 教室式座位（有桌子）的最大容納人數
    - 數字

15. **平日時間**（選填）
    - 平日可使用的時間
    - 例如：「09:00-21:00」

16. **假日時間**（選填）
    - 假日可使用的時間
    - 例如：「09:00-18:00」

17. **設備費用**（選填）
    - 可提供的設備及費用
    - 例如：「投影機 NT$500/次, 音響 NT$1000/次」

18. **備註**（選填）
    - 其他重要資訊
    - 例如：「需提前一週預約」、「附停車場」

19. **活動規模**（自動判斷）
    - 根據「最大容納(空場)」自動分類：
      - 微型活動：≤50人
      - 小型活動：51-100人
      - 中型活動：101-200人
      - 中大型活動：201-500人
      - 大型活動：501-1000人
      - 超大型活動：1001-5000人
      - 巨型活動：>5000人

## ✅ 資料品質要求

### 必須符合：
1. **真實性**：所有場地必須是真實存在且可租借的
2. **完整性**：必填欄位必須完整填寫
3. **準確性**：聯絡資訊必須準確可聯繫
4. **最新性**：資訊應為近期的資料
5. **多樣性**：涵蓋不同縣市、不同類型、不同規模

### 資料來源驗證：
- 優先使用官方網站資訊
- 民營場地需有官方網站或粉絲頁
- 聯絡電話需可打通
- 地址需可對應到實際地點

## 🎯 收集目標

### 數量目標：
- **第一階段**：每個縣市至少 5 個場地（共 110 個）
- **第二階段**：每個縣市至少 10 個場地（共 220 個）
- **第三階段**：重點城市（六都）每個至少 20 個場地

### 品質目標：
- 必填欄位完整率 > 90%
- 聯絡資訊正確率 > 80%
- 價格資訊完整率 > 60%

## 💡 提示建議

### 搜尋技巧：
1. 使用 Google 進階搜尋：`site:gov.tw 場地租借`
2. 搜尋特定平台：`site:accupass.com 場地`
3. 搜尋 PDF 檔案：`filetype:pdf 場地租借費用`
4. 使用地圖搜尋：Google Maps「附近會議室」

### 資料整理：
1. 先收集網頁連結
2. 再逐一提取資訊
3. 驗證聯絡資訊
4. 整理成 CSV 格式

### 常見問題處理：
- **沒有價格**：留空或填「需詢價」
- **價格區間**：取平均值或填「3000-5000」
- **多個廳別**：每個廳別一筆資料
- **沒有電話**：填 Email 或官網連結
