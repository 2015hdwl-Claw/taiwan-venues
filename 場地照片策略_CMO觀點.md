## 三、緊急方案：佔位圖設計策略

### ❌ 目前的問題
```html
<div class="venue-image-placeholder">🏢</div>
```

只有一個 emoji，看起來像：
- 測試網站
- 未完成產品
- 不專業平台

### ✅ 建議方案：三層佔位圖策略

#### 層級 1：基礎佔位圖（今晚實施）

**設計原則**：
- 品牌一致性
- 場地類型識別
- 專業感
- 不讓用戶覺得「壞掉了」

**設計方案**：

```css
/* 根據場地類型顯示不同的佔位圖 */
.venue-image-placeholder {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    display: flex;
    flex-direction: column;
    align-items: center;
    justify-content: center;
    gap: 8px;
}

.venue-image-placeholder::before {
    content: attr(data-type-icon);
    font-size: 48px;
}

.venue-image-placeholder::after {
    content: "照片籌備中";
    font-size: 14px;
    color: rgba(255,255,255,0.9);
    font-weight: 500;
}

/* 不同類型的漸層色 */
.venue-image-placeholder[data-type="飯店"] {
    background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
}

.venue-image-placeholder[data-type="大學"] {
    background: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);
}

.venue-image-placeholder[data-type="展覽場地"] {
    background: linear-gradient(135deg, #43e97b 0%, #38f9d7 100%);
}
```

**HTML 修改**：
```html
<div class="venue-image-placeholder" 
     data-type="${venue.venueType || '會議場地'}"
     data-type-icon="${getVenueIcon(venue.venueType)}">
</div>
```

**Icon 對照表**：
```javascript
function getVenueIcon(type) {
    const icons = {
        '飯店': '🏨',
        '大學': '🎓',
        '展覽場地': '🎨',
        '會議中心': '🏛️',
        '文創園區': '🎭',
        '體育場館': '🏟️',
        '政府機關': '🏢',
        '商務中心': '💼',
        'default': '🏢'
    };
    return icons[type] || icons.default;
}
```

**效果**：
- ✅ 品牌漸層色（視覺統一）
- ✅ 類型識別（一眼看出是飯店/大學/展覽場地）
- ✅ 專業文案（「照片籌備中」而非空白）
- ✅ 不像「壞掉了」

## 四、長期策略：如何取得場地照片

### 🎯 目標
3 個月內達成 **95% 照片覆蓋率**

### 策略 1：官方合作（最理想）

**執行方式**：
1. **聯繫場地方**（公關公司優勢）
   - 準備標準信件範本
   - 強調「免費曝光」
   - 提供「合作夥伴」標章

2. **交換價值**：
   - 場地方提供：高品質照片
   - 活動大師提供：
     - 免費刊登
     - 優先排序
     - AI 推薦權重 +20%
     - 「官方認證」標章

3. **分批執行**：
   - 第一週：前 50 大場地（飯店、大學）
   - 第二週：中型場地
   - 第三週：小型場地

**預期成果**：
- ✅ 高品質官方照片
- ✅ 建立合作關係
- ✅ 未來媒合基礎

---

### 策略 2：公開資源（快速）

**來源**：
- **Wikipedia Commons**: 知名地標（如國家歌劇院）
- **場地官方網站**: 爬蟲抓取（需注意版權）
- **Google 地圖**: 用戶上傳照片（需授權）

**執行步驟**：
1. Jobs 技術團隊建立照片爬蟲
2. 自動檢查版權狀態
3. 標註來源

**優點**：快速
**缺點**：品質參差不齊

---

### 策略 3：用戶貢獻（長尾）

**設計 UGC（User Generated Content）機制**：

```
📸 上傳場地照片，賺取點數！

上傳 1 張照片 = 10 點
上傳 5 張照片 = 50 點 + 優先諮詢權
照片被採用 = 100 點 +「貢獻者」標章

點數可兌換：
- 場地預約折扣
- 免費活動諮詢
- VIP 服務
```

**優點**：
- ✅ 長期解決方案
- ✅ 真實照片（信任度最高）
- ✅ 用戶參與感

**缺點**：
- ❌ 需要審核機制
- ❌ 初期效果慢

---

### 策略 4：實地拍攝（高品質）

**針對 top 50 場地**：
- 聘請攝影師
- 實地拍攝
- 包含：外觀、內部、設備、座位配置

**預算**：
- 約 2,000-5,000 元/場地
- 總預算：10-25 萬

**ROI 計算**：
- 如果 top 50 場地佔 80% 預約量
- 值得投資
