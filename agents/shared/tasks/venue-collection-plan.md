## 🔧 收集方法

### 1. 官方網站
- 各飯店官網的「場地租借」頁面
- 政府機關的場地租借資訊

### 2. 線上平台
- EZ訂場地
- Fun心訂
- 場地王

### 3. AI 輔助
- 使用 Grok 搜尋「XX縣市 場地租借」
- 使用 Grok 提取場地詳細資訊

### 4. 電話確認
- 確認費用、容納人數等關鍵資訊
- 索取照片和官方網站

## 📝 資料格式

```json
{
  "name": "場地名稱",
  "roomName": "廳別",
  "venueType": "飯店場地/會議場地/展演場地/機關場地",
  "city": "縣市",
  "address": "詳細地址",
  "contactPhone": "電話",
  "contactEmail": "Email",
  "priceFullDay": 全天費用（數字）,
  "maxCapacityTheater": 容納人數（數字）,
  "images": {
    "main": "主照片URL",
    "gallery": ["其他照片URL"]
  },
  "url": "官方網站URL"
}
```
